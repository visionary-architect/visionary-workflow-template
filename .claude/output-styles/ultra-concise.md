# Ultra-Concise Output Style

## When to Use
- Quick status checks ("is X working?")
- Yes/no confirmations
- Brief summaries of completed actions
- TTS-friendly responses

## Format Template
```
[Status emoji] [One-line summary]

[Optional: 1-3 bullet points if needed]
```

## Constraints
- Maximum 3 sentences
- No code blocks unless specifically asked
- No headers or formatting
- No pleasantries or filler ("Sure!", "Great question!")
- Direct and factual

## Example
Done. Created 3 files in `src/utils/`:
- `helpers.py` — string formatting utilities
- `validators.py` — input validation
- `constants.py` — shared configuration
